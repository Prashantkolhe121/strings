- Function - It is the relation between Input & Output.
- Function - Block of code which is created to perform a particular task.
- Function is very similar to Mathematical formulas.
- Ex. Fruit-Juice-Machine can be a function which is created to perform a particular task which inputs the fruits & gives the juice of that fruit. So the content (any fruit) is the Parameter - local declaration, & machine has been instructed to make the juice of any fruit ( Parameters - local declaration ) it will receive in the input. To make the juice is the operation / function which we have defined. Juice is the Output of this function that we will receive after calling the function with the right Arguements. Arguements is the actual input values we receive when the function is being called. Arguement can be an Apple, Banana, Orange. So the Output / Juice ( If the input is apple then you will get apple juice ) will be the result which will vary on the basis of Input. But the operation / process / function would remain same in any case.
- Parameters & Arguments in Function - Local declarations / variables which is used at the time of function declaration which is limited for function block only & used for that function operation or task. However, Arguments are the declarations or variables which is used at the time of function call & it's the actual Input values to be used for the operation / task.
Ex.
function addTwoNum (firstNum, secondNum) { // firstNum & secondNum are the parameters
var resultOfTwo = firstNum + secondNum;
// You may store the result in a variable console.log(resultOfTwo); // This will be the output of this function when it's called
// return firstNum + secondNum;
// or You may ' return '' it, so it can be fetched easily when this function is called
}
// var resultofTwo = addTwoNum(73,89); // function is called & result is stored in the variable resultOfTwo
